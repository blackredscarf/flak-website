Map and Set

Map and Set are associative containers. They can be used to store arbitrary elements, such as integers, strings or custom classes.

Map

The Map containers have Red Black Tree Map, Hash Map, and AVL Tree Map. Their interfaces are the same as the STL map.

The header follows:

#include <flak/AVLMap.h>     // avl tree map
#include <flak/HashMap.h>   // hash map
#include <flak/Map.h>       // red black tree map

The interfaces:

flak::AVLMap<string, int> avlmap;
flak::HashMap<string, int> hashmap;
flak::Map<string, int> rbmap;

Examples:

#include<flak/AVLMap.h>
#include <iostream>
using namespace std;

int main() {
    // The interface is the same as the STL map
    flak::AVLMap<string, int> smap;
    smap["jjhou"] = 1;
    smap["jerry"] = 2;
    smap["jason"] = 3;
    smap["jimmy"] = 4;

    flak::AVLMap<string, int>::iterator it1 = smap.begin();
    for(; it1 != smap.end(); it1++) {
        cout << it1->first << " : " << it1->second << endl;
    }

    smap.erase("jason");
    it1 = smap.find("jason");
    cout << (it1 == smap.end()) << endl; // true
}

Set

The Set containers have Red Black Tree Set, Hash Set, and AVL Tree Set. Their interfaces are the same as the STL set.

The header files:

#include <flak/AVLSet.h>    // avl tree set
#include <flak/HashSet.h>   // hash set
#include <flak/Set.h>       // red black tree set

The interfaces:

flak::AVLSet<string, int> avlset;
flak::HashSet<string, int> hashset;
flak::Set<string, int> sset;

Examples:

#include <flak/AVLSet.h>
#include <iostream>
using namespace std;

int main() {
    int ia[5] = {0, 1, 2, 3, 4};
    flak::AVLSet<int> iset(ia, ia+5);
    cout << "size: " << iset.size() << endl;

    // insert
    iset.insert(11);
    cout << "size: " << iset.size() << endl;

    // count
    cout << "count: " << iset.count(3) << endl;

    // erase
    iset.erase(1);
    auto it = iset.find(1);
    cout << (it == iset.end()) << endl;

    // iterator
    auto it1 = iset.begin();
    for(; it1 != iset.end(); it1++) {
        cout << *it1 << " ";
    }
    cout << endl;
}
