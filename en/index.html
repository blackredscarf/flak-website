Flak is a general and efficient data structures and algorithm template library. Implementation in STL style and easy-to-use interface.

Flak includes three modules:

Container: A container is a holder object that stores a collection of other objects (its elements). They are implemented as class templates, which allows great flexibility in the types supported as elements. Such as Map, Set, Vector, List, KDTree, Trie and so on.

Algorithm: A collection of functions especially designed to be used on ranges of elements. A range is any sequence of objects that can be accessed through iterators or pointers, such as an array or an instance of some of the containers. Such as Sort, Random, Heap and so on.

Graph: A library contained the graph data structures and algorithms. Such as AdjacencyList, Dijkstra, BFS, DFS, DisjointSet and so on.
